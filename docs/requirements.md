# 媒体管理器（WinUI 3）功能需求文档

> 目标：从零新建 Windows-only 媒体播放器与管理器，强调现代化 UI、性能流畅与稳定。

## 1. 产品目标
1. Windows-only 媒体播放器与管理器，强调现代化 UI、快速响应、稳定播放与高效整理。
2. 支持视频与图片统一管理与检索，能够对媒体进行标签化与播放列表组织。
3. 支持基本视频剪辑（保留/删除片段），并且具备可视化进度与状态反馈。

## 2. 核心用户场景
1. 选择文件夹或文件导入媒体库，自动识别媒体类型并生成缩略图。
2. 通过搜索与标签筛选媒体，快速定位目标内容。
3. 点击媒体直接播放或查看，支持快捷键控制播放与切换。
4. 批量选择媒体进行删除或统一编辑标签。
5. 对视频进行区间剪辑并导出新文件。

## 3. 功能需求（详细）

### 3.1 媒体库管理
1. 支持导入媒体文件（多选）与导入文件夹（多选）。
2. 支持拖拽文件/文件夹到媒体库导入。
3. 扫描文件夹时递归子目录，支持常见视频与图片格式。
4. 扫描过程中显示进度与批量增量导入。
5. 支持“刷新媒体库”，对已监控文件夹重新扫描并增量更新。
6. 媒体元数据字段：
   1) 路径、文件名、大小、创建/修改时间
   2) 宽高、视频时长
   3) 缩略图

### 3.2 标签系统
1. 每个媒体可添加多个标签。
2. 标签可编辑与删除。
3. 多选媒体时可统一编辑标签。
4. 标签搜索支持“必须包含所有标签”。
5. 搜索框支持输入 `#标签` 快速加入标签筛选。

### 3.3 搜索与排序
1. 搜索词匹配文件名与标签。
2. 可按文件名、修改时间排序，支持升序/降序循环切换。
3. 支持列表视图与网格视图切换。
4. 网格视图支持 `Ctrl + 滚轮` 调整图标尺寸。

### 3.4 播放体验（视频）
1. 视频播放支持主流格式。
2. 自定义控制条固定在底部：播放/暂停、进度、音量、全屏、时间显示。
3. 鼠标移动显示控制条，闲置自动隐藏。
4. 播放模式：列表循环 / 单曲循环 / 随机播放。
5. 播放完成自动跳转符合播放模式规则。

### 3.5 图片查看
1. 图片查看支持缩放与拖拽平移。
2. 滚轮/快捷键缩放（+ / - / 0 重置）。
3. 支持前后切换。

### 3.6 播放列表
1. 支持创建、重命名、删除播放列表。
2. 支持拖拽排序播放列表顺序。
3. 支持将媒体添加到播放列表（新建或已有）。
4. 播放列表内媒体可移除，不影响原文件。
5. 播放列表媒体支持搜索与排序。
6. 支持播放列表独立窗口（可选）。

### 3.7 视频剪辑
1. 支持两种模式：保留模式 / 删除模式。
2. 支持多段区间编辑，区间可拖拽微调。
3. 支持手动输入开始/结束时间。
4. 支持选择输出目录。
5. 剪辑过程显示进度与状态。

### 3.8 界面与窗口
1. 深色主题、现代化视觉语言。
2. 侧边栏支持固定/自动隐藏。
3. 侧边栏宽度可拖拽调整。
4. 右键菜单覆盖主播放区与媒体列表。

### 3.9 设置
1. 数据目录可配置。
2. 便携模式：数据固定到安装目录 `data`。
3. 缓存管理：
   1) 清理缩略图与临时文件
   2) 清理缓存并重置库
   3) 清理全部应用数据（高风险）
4. 监控文件夹管理：添加/移除/清除。
5. 文件夹锁定：全局密码、锁定/解锁状态。

### 3.10 快捷键
1. `Ctrl + O` 添加文件夹
2. `Ctrl + F` 聚焦搜索
3. `Space` 播放/暂停
4. `← / →` 后退/前进 5 秒
5. `Page Up / Page Down` 上一个/下一个
6. `Delete` 删除选中
7. `Ctrl + T` 添加 Tag
8. `Esc` 关闭弹窗

## 4. 数据模型与持久化
1. 数据库：SQLite。
2. 媒体表字段：
   1) `id`、`path`、`filename`、`type`、`size`
   2) `width`、`height`、`duration`
   3) `thumbnail`、`createdAt`、`modifiedAt`
3. 标签表：`mediaId` + `name` + `createdAt`。
4. 播放列表表：`id`、`name`、`sortOrder`、`createdAt`。
5. 播放列表媒体关联表：`playlistId`、`mediaId`、`sortOrder`、`addedAt`。

## 5. 非功能需求
1. 扫描与缩略图生成需避免阻塞 UI。
2. 任何 IO/剪辑操作需具备错误处理与提示。
3. 需要稳定处理大批量媒体，列表使用虚拟化。
4. 缓存清理、扫描、剪辑等操作具备明确状态反馈。
