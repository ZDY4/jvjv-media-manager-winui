# MainPage 回归清单

## 库区

- 导入文件夹、导入文件、拖放导入都能刷新媒体库，并将新路径纳入监视目录。
- 搜索框输入文本会在短暂防抖后更新结果。
- 在搜索框输入 `#标签` 并回车会追加标签过滤，并清空搜索输入框。
- 删除标签筛选后，结果集会即时恢复。
- 媒体库 / 播放列表切换时，左侧 Pane 展开与收起行为符合预期。
- 列表 / 网格切换后，当前选中项仍可保持并滚动到可见区域。
- 按 `Ctrl + 鼠标滚轮` 时，列表缩略图和网格卡片尺寸会同步变化。
- 排序按钮会在“名称升序 -> 名称降序 -> 时间升序 -> 时间降序”之间循环。

## 播放列表

- 新建播放列表后会自动保留在展开态，并可在左侧栏立即看到新项。
- 右键播放列表可以重命名、改色、删除。
- 删除当前播放列表后会回到“全部媒体”作用域。
- 拖拽播放列表排序后，重启应用仍保持顺序。

## 播放器

- 选择视频时会隐藏空态、显示信息浮层，并进入视频控制栏模式。
- 选择图片时会显示缩放角标，并支持滚轮缩放、双击复位、拖拽平移。
- 清空选择、删除当前媒体或筛选后无选中项时，会回到播放器空态。
- `PageUp / PageDown` 可以切换上一个和下一个媒体。
- 右键播放器区域时，媒体上下文菜单与库区右键菜单保持一致。

## 快捷键

- `Ctrl + O` 导入文件夹。
- `Ctrl + F` 展开库区并聚焦搜索框。
- `Ctrl + T` 打开标签编辑。
- `Delete` 删除当前选中媒体。
- 视频状态下 `Space` 播放 / 暂停，`Left / Right` 快进快退。
- 图片状态下 `+ / - / 0` 对应放大、缩小、复位。

## 对话与剪辑

- 标签编辑、设置、锁定管理、播放列表选择对话能正常打开。
- 剪辑模式下 `I / O / E` 快捷键仍可正常工作。
- 导出剪辑成功后，媒体库状态和输出目录提示正常刷新。
